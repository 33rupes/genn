	def is_model_valid()

Check whether the model is valid, i.e is native or derived from model_family.Custom.

Raises ValueError if model is not valid (i.e. is not custom and is not natively available)



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- model

		- string or instance of model_family.Custom

	*
		- model_family

		- model family (:ref:`NeuronModels <doxid-d1/d5d/namespace_neuron_models>`, :ref:`WeightUpdateModels <doxid-dd/d76/namespace_weight_update_models>` or :ref:`PostsynapticModels <doxid-db/d7d/namespace_postsynaptic_models>`) to which model should belong to

instance of the model and its type as string
